About

This playground shows how to run nginx as a static file server in Docker with production-relevant configuration.

Location block types

location / Prefix match — catches all paths, uses try_files
location = /health Exact match — fastest, used for fixed paths like health checks
location ~* \.(css|js|...)$ Regex match (case-insensitive) — targets static asset extensions

Response headers

X-Content-Type-Options Prevents MIME-type sniffing
X-Frame-Options Blocks clickjacking via iframes
Referrer-Policy Controls how much referrer info is sent with requests
Cache-Control public, immutable on static assets — CDN and browser friendly
