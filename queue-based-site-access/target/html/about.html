How the virtual waiting room works

🏗️ Architecture

A Go service sits in front of a protected Nginx page. Every incoming request passes through a session gate — users without an active slot are placed in a Redis-backed FIFO queue.

🔒 Bypass prevention

The protected page has no published port in Docker Compose. There is no TCP path from a browser to the target without going through the queue service — bypassing is architecturally impossible.

💓 Session keepalive

JavaScript pings a heartbeat endpoint every 10 seconds. A background reaper evicts sessions that stop sending heartbeats, freeing their slot for the next person in line within ~35 seconds.
