Features

What this system provides out of the box

  • ⚡
    Atomic slot claiming

    Redis Lua scripts ensure two concurrent visitors can never claim the same last slot — check and write are a single atomic operation.

  • 📊
    Live position & ETA

    The queue page polls every 2 seconds and shows your position, total queue length, and an estimated wait time once 5 admissions have been recorded.

  • 🔄
    Refresh-safe sessions

    Refreshing the protected page or the queue page never evicts your session. Positions and active slots are preserved across reloads.

  • 🛠️
    Admin API

    Change capacity on the fly, release active slots, and inspect live queue state — all via simple JSON endpoints.
